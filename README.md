##  Topics Covered So Far
* **Basic I/O:** `printf()` for formatted output and `scanf()` for smooth user input handling.
* **Data Types & Arrays:** Working with integers (`int`), floating-point numbers (`float`), characters (`char`), and sequential integer arrays.
* **Type Casting:** Explicitly converting data types to prevent integer division truncations and maintain system precision.
* **Conditional Logic & Operators:** Implementing robust multi-way branching (`if-else if-else`) and harnessing logical operators (`||`, `&&`) for precise condition matrix evaluation.
* **Fault-Tolerant Loops & Recoveries:** Designing boundary protection loops with explicit failure thresholds to handle invalid runtime states and guarantee system safety.
* **Algorithmic Optimizations:** Constraining loop iteration boundaries to optimize variable spaces and reduce CPU cycle consumption on predictable data sets.
* **Mathematical Property Scanners:** Creating algorithms to dissect and identify mathematical structures including Divisors, Primality, and Common Factors.
