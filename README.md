##  Topics Covered So Far
* **Basic I/O:** `printf()` for formatted output and `scanf()` for smooth user input handling.
* **Data Types:** Working with integers (`int`), floating-point numbers (`float`), and characters (`char`).
* **Strings:** Handling basic character arrays and printing strings using `%s`.
* **Arrays:** Defining, initializing, and modifying integer arrays (`int numbers[10]`).
* **Type Casting:** Explicitly converting data types (e.g., `(float) i2 / i1`) to prevent integer division errors and maintain precision.
* **Conditional Logic:** Implementing robust multi-way branching using `if`, `else if`, and `else` statements.
* **Logical Operators:** Harnessing logical OR (`||`) to create multi-condition evaluation and handle case-insensitive user inputs (e.g., matching both 'r' and 'R').
* **Input Validation:** Creating fallback mechanisms to handle invalid user configurations and gracefully log errors to the console.
