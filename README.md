# My C Learning 

This repository contains my daily C programming learning journey.

## Purpose
* Practice C basics step by step
* Track my progress over time
* Save small exercises and experiments

## Structure
* `learning/` → small practice codes
* `projects/` → bigger programs (later)

## Topics covered so far
* **Basic I/O:** `printf()` for formatted output and `scanf()` for user input.
* **Data Types:** Working with integers (`int`), floating-point numbers (`float`), and characters (`char`).
* **Strings:** Handling basic character arrays and printing strings using `%s`.
* **Arrays:** Defining, initializing, and modifying integer arrays (`int numbers[10]`).
